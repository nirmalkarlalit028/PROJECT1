    • DATA-TYPE

    INFORMATION:- In JavaScript, data types define the kind of values a variable can hold. The language is dynamically typed, meaning you don't need to declare a variable's type; it's determined automatically at runtime based on the value assigned.

    1. Primitive Data Types
    Number: Represents both integers and floating-point numbers (e.g., 42, 3.14). It also includes special values like Infinity and NaN (Not-a-Number).
    String: Textual data enclosed in single ('), double ("), or backticks (`).
    Boolean: A logical type that can only be true or false.
    Undefined: A variable that has been declared but not yet assigned a value.
    Null: Represents an intentional absence of any value.

    2. Non-Primitive (Reference) Data Type
    Object: Used to store collections of data and more complex entities. Arrays, Functions, and Dates are all technically specialized types of objects in JavaScript. Unlike primitives, objects are mutable (can be changed) and are stored by reference in memory.
