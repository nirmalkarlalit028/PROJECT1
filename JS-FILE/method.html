    • Method

    INFORMATION:- A method in JavaScript is a function that is stored as a property of an object. It defines the behavior or actions an object can perform, operating on the data contained within that object.

    1. Method of String
    1. Uppercase: In JavaScript, the main way to convert a string to uppercase is the toUpperCase() method. It's a built-in function for all string values ​​and returns a new string (converting all lowercase (small) letters to uppercase (capital) letters) without changing the original string, because JavaScript strings are immutable.
    2. Lowercase:In JavaScript, the main way to convert a string to lowercase is the toLowerCase() method. It's a built-in function for all string values ​​and returns a new string (converting all small/capital letters to small letters) without changing the original string, because JavaScript strings are immutable.
    3. Trim: The JavaScript trim() method removes whitespace from both the beginning and end of a string. It returns a new string and does not modify the original one.
    4. CharAt: The charAt() method in JavaScript returns a new string consisting of the single character at the specified index within the original string.
    5. Include: The includes() method in JavaScript is a built-in function used to determine whether an array contains a specific element or a string contains a specific substring. It is case-sensitive and returns a boolean value (true or false).
    6. Concat: The concat() method in JavaScript is used to merge two or more strings or arrays into a single, new entity. It is a non-mutating method, meaning it does not change the original strings or arrays, but instead returns a shallow copy containing the combined elements.
    7. Slice: The JavaScript slice() method is used to extract a portion of an array or a string and return it as a new array or new string, respectively, (this method zero-based index, startIndex & endIndex) without modifying the original source. This process creates a shallow copy of the selected elements.

    2. Method of Array
    1. Unshift: The JavaScript unshift() method adds one or more elements to the beginning of an array and returns the new length of the array. It modifies the original array in place.
    2. Push: The JavaScript push() method is used to add one or more elements to the end of an array. It modifies the original array in place and returns the new length of the array after the elements have been added.
    3. Shift: The shift() method in JavaScript is an array method that removes the first element from an array and returns that removed element. This method modifies the original array and reduces its length by one.
    4. Pop: The JavaScript pop() method is used to remove the last element from an array and returns that removed element. This method is a mutating method, meaning it changes the original array's length and content.
    5. Foreach: The foreach() method in JavaScript is an array iteration method used to execute a provided function once for each element in an array. It is primarily used for performing a side effect (e.g., logging to the console, updating the DOM, summing values) rather than creating a new array or returning a value. The method returns undefined. (this method not return value)
    6. Map: The map() method creates a new array populated with the results of calling a provided callback function on every element in the original array. A crucial aspect is that it does not modify the original array. (this method return the value)
    7. Filter: The filter() method is used to create a new array containing only the elements from the original array that pass a specific condition implemented by a provided callback function. It does not modify the original array.
